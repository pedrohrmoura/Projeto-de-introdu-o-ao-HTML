HTML

O HTML foi criado por um físico britânico chamado Tim Berners Lee, HTML é uma linguagem de marcação que é utilizada principalmente para a construção de sites na internet.

HTML é uma abreviação da expressão HyperText Markup Language, que significa "Linguagem de Marcação de Hypertexto"

Tags

Tags são elementos que servem diferentes funções dentro de um site HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros.

Tags e suas funções

html

Representa a raiz do do site, serve com um container que engloba todos os outros elementos HTML

head

Compreende as informações do documento que serão interpretadas pelo navegador. Como, por exemplo, título do documento, links para folhas de estilo, etc.

body

É onde fica todo o conteúdo de texto, imagem e vídeos, em que o usuário ou usuária vê e interage, nele os conteúdos são estruturados pelas demais tags do HTML.

h1, h2, h3 h4, h5, h6, p

São tags utilizadas para definir textos como títulos e parágrafos dentro do site

strong, i, mark, sup, blockquote, u

São tags utilizadas para mudar o texto como por exemplo colocar em negrito, itálico, marcado ou sublinhado.

ol, ul

São tags utilizadas para realizar listas ordenadas e não ordenadas respectivamente, exemplo:

  1. Primeiro
  2. Segundo
  3. Terceiro
  • Primeiro
  • Segundo
  • Terceiro

a, href
